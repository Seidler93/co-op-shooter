# 🎮 Launcher & Game – Style Guide

## 1. Core Vision

This launcher and game should feel:

* Premium
* Tactical
* Slightly gritty
* Clean and modern
* Performance-focused (not bloated or flashy)

Think:

* Military UI meets modern AAA game launcher
* Clean like Steam, but more immersive
* Slight sci-fi tone without being over-the-top

---

## 2. Visual Tone & Mood

### Keywords

* Dark
* Focused
* Tactical
* Minimal
* High contrast
* Subtle glow accents

### Avoid

* Cartoonish UI
* Overly bright colors
* Cluttered layouts
* Excessive animations

---

## 3. Color Palette

Primary:

* Background: #0D0F14 (near black)
* Panels: #151922
* Elevated panels: #1C2230

Accent:

* Primary accent: #4D7CFF (blue)
* Secondary accent: #7CF5A3 (green success / active)
* Danger: #FF5C5C

Text:

* Primary: #FFFFFF
* Secondary: #A0A7B8
* Muted: #6B7280

---

## 4. Typography

Style:

* Clean sans-serif
* Slightly condensed for UI density

Hierarchy:

* Large headers: Bold, spaced
* Section titles: Medium weight
* Body text: Light/regular
* Buttons: Medium + uppercase

---

## 5. Layout Rules

* Use card-based panels
* Rounded corners (8px–16px)
* Consistent spacing (8px grid system)
* Strong alignment (no floating/random elements)

Main Layout:

* Header (top)
* Content (center)
* Primary action (bottom-left)

---

## 6. Components

### Buttons

* Rounded
* Slight glow on hover
* Clear states:

  * Default
  * Hover
  * Disabled
  * Active

### Panels

* Slight elevation (shadow or lighter background)
* Subtle border or glow

### Dropdowns

* Smooth open animation
* Dark theme with hover highlights

---

## 7. Interaction Feel

* Fast and responsive
* Subtle transitions (150–250ms)
* No blocking UI unless necessary
* Feedback on every action

---

## 8. Launcher-Specific UI

### Main Dashboard

* Bottom-left: Primary action (Download / Play / Update)
* Center: Game info + patch notes
* Top-right: Profile + settings dropdown

### Beta Access

* Locked state should feel intentional (not broken)
* Prompt should feel like unlocking something exclusive

---

## 9. Inspirations

Use these as references when designing:

* Battle.net Launcher (clarity + spacing)
* Escape from Tarkov UI (tone + mood)
* Call of Duty menus (polish + flow)

---

## 10. Future Art Direction (Game)

* Realistic but slightly stylized
* Dark environments with strong lighting contrast
* Clear silhouettes for enemies and players
* Minimal UI clutter during gameplay

---

## 11. Notes for Development

* Always prioritize clarity over style
* Every UI element must have a purpose
* Avoid adding UI just for aesthetics
* Keep everything scalable for future features
