FAQ

Beta Notes

Will I keep using this forever?

No. Once the game is ready for public release, distribution can move to Steam.

Do I need a new download for each update?

No. The launcher checks for updates automatically and pulls the latest approved build.

Can anyone install it?

Only players with a valid invite code and account access can install the beta build.
